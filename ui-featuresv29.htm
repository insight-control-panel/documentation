UI-FeaturesV29
❑Layout
❑Color and font -> bold and italic;  attributes can have color and font
❑List -> Multi line list entries + filter (replace List-Attributes)
❑ListEdit ->  Multiple records can be edited on one page.
❑Control -> hidden, disabled or createOnly
❑Floating -> Attributes can be next to each other
❑LabelHidden -> Only the input control is visible
❑Menu actions
❑goto -> Can route to list edit views
❑listEdit-Menu -> Routes to list edit views
❑online -> sets the app online if the server is available
❑offline -> sets the app offline
❑Search Features
❑clearSearch -> Removes the search text when the page is entered.
❑searchPattern -> Automatically add * wildcards while searching
