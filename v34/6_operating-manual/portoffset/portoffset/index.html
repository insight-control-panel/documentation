    port-offset¶

    Standard-port for Wildfly is 8080.

    If its neseccary to change the standardport then it is adviced to adjust property jboss.socket.binding.port-offset in standalone.xml and set this value > 0 i.e port-offset=100 shifts all ports of wildfly to a value +100 and wildfly now can be reached on Port 8180
