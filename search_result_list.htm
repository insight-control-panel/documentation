Search Result List

Via each attributes clientProperties there's control over wether the attribute is hidden or shown in every search result, no matter if there search stringed matched or not.

 

Example:

 

"attributes": [{

 "name": "DESCRIPTION",

 "label": "Zusammenfassung",

 "type": "String",

 "clientProperties": {

         "search": {

                 "inResult": "show"

         }

 }

}, {

 "name": "SYSTEMID",

 "label": "System",

 "type": "String",

 "clientProperties": {

         "search": {

                 "inResult": "hide"

         }

 }

}, { .... }]
