Installation Insight Document

ATTENTION: insight-document, insight-document-handler-maximo and insight-document-handler-file are removed. All is integrated in insight-server. A new insight property "document.handler" was added.

ATTENTION: Instead of the use of "document.handler", the handler can be configured in the documents configuration.

❑Insight Document is a Middleware-Module to upload documents to the server. The insight-mobile-app can upload images to the server.
❑These are two implementations of the document upload. You can only use one handler. (maximo is default)
ofile handler
omaximo handler
oinsight handler
❑Document-Configuration in Tree-Configuration:
oThis can be configured at a node:

"documents": [{

 "handler": "file",

 "name": "foto",

 "label": "Fotos",

 "type": "Foto",

 "backendParams": {

         "OWNERTABLE": "WORKORDER",

         "OWNERID": "${WORKORDERID}",

         "SITEID": "${userData.siteid}"

 }

}]

o"label" is shown on the UI
o"type"
▪"Foto"  Capture or choose a photo. Supported on iOS and Android
▪"File"  Choose a file. Supported on Electron
oIn "backendParams" you can configure properties, which are useful for your backend
o"handler" to use (maximo, file or insight)
o"clientProperties.headerHidden" Boolean value to hide the document section header.

"documents": [{

 ...

 "clientProperties": {

   "headerHidden": true

 }

}]

o"clientProperties.inlineDelete" Boolean value to show a delete button on the image. The image can also be deleted within the state popup dialog.

"documents": [{

  ...

  "clientProperties": {

    "inlineDelete": true

  }

}]

o"clientProperties.imageEdit" Photos can be modified before upload, by default. This can be changed with the imageEdit setting. The edited image replaces the original, this can be changed using "copy"
▪"disabled" Boolean value to hide the imageEdit action icon.

"documents": [{

  ...

  "clientProperties": {

    "imageEdit": {

      "disabled": true

    }

  }

}]

▪"copy" Boolean value to copy the image after editing.

"documents": [{

  ...

  "clientProperties": {

    "imageEdit": {

      "copy": true

    }

  }

}]
