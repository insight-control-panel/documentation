V13.3.1
Added duration widget: Input widget for submitting time durations.
❑searchReplaceFunction: Executes a script before sending the search string to the search api. Useful to modify barcode scan results.
❑Menu - swipeHidden: Hides the menu action in insight-mobile's list swipe.
❑Added CSV Export for Insight Tables: Exports the current selection of data from table to a CSV file.
❑Fullscreen support: Several Insight components can now be displayed in fullscreen. This includes attributes, tables and the file viewer of the details page as well as gantt charts.A fullscreen button is displayed at the top right of each component, next to the collapse button.
❑Gantt multi monitor support. A gantt chart can now be opened in a new window allowing for better usability when using the Insight planning feature. Drag and Drop between a table on one browser window and the gantt chart on the second browser window is supported.
❑qrCode action: Renders a QR Code clientside based on the given configuration for other users to use as a deepLink.
❑barcodeDeeplink: Scans a barcode and handles it's content as a deepLink.
❑clear is now supported in web as well
❑Default- and Fallback language are configurable
❑toolstips for menuActions are configurable.
❑The duration Widget supports negative values and can be configured with custom conversion factors.
❑The default behaviour for the table's selectedItems feature changed. You know need to explicitly activate this feature by setting useSelectedItems: true in the table's settings.
