    Dynamic Combo box¶

    Combo- or Dropdown-Boxes have to be, by default, configured with a static dictionary of keys and values.

    You can load this dictionary from a script with this configuration:

    {
    "attributes": [
    {
        "label": "Team",
        "name": "resourceId",
        "type": "String",
        "control": "enabled",
        "section": "Team-Auswahl",
        "widget": "Combo",
        "clientProperties": {
            "attributeControl": {
                "js": [
                    "remote.action('business-suite/wka/teams-list', {})",
                    "  .then(",
                    "    function (data) { attribute.options = data.options; }",
                    "  )"
                ]
            }
        },
        "options": {
            "dictionary": []
        }
    }
    ]
}

    The corresponding script has to return a list of key:value objects:

    // Returns list to be shown in dropdown of a form-dialog

def options = []

sql.eachRow("""
    SELECT
        id,
        name,
        description
    FROM Resource as team
    WHERE team.classificationName = 'TEAM' AND team.siteName = 'WKA'
""") { team ->
    String id = team[0]
    String name = team[1]
    String description = team[2]
    options.push([label: name, value: id, description: description])
}

return jsonObject(options: options)
