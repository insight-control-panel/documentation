Kibana

You can access Kibana with your browser on default port 5601 of your Kibana server.

Please make sure that the attributes you want to analyze are defined in an Insight Tree configuration. Information defined in “Insight tree configuration” map to the records queried by Kibana from Elasticsearch.

Insight Configuration

Elasticsearch/ Kibana record

Name of Insight Configuration

Value of property “_index”

Name of root or children

Value of property “_type”

Name of Attribute

Property “properties.<name of attribute>” will contain the value of the attribute.

Label of Attribute

The label of the attribute is currently not visible in Kibana. Please use your Insight configuration for reference in later configuration steps.

The following excerpt from a tree configuration shows an example:

{
 "name": "organigramm",              à _index = "organigramm"
 "root":{
   "name": "Person",            à _type = "Person"
   "attributes": [{
     "name": "508:40325:AB"        à properties.508:40325:AB =

                                   <last name of Person>
     "label": "last name"        à (not visible in Kibana)
     …
   }],
   "children":[{
     "name": "delivery address",        à _type = "delivery address"
     "attributes": [{
       "name": "508:40335:AB"        à properties.508:40335:AB =
                                         <city of delivery address>
       "label": "city"                à (not visible in Kibana)
       …
   }],
   …
 },
 …
}
