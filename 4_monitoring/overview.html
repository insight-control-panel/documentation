    Overview

    This chapter describes how to monitor the insight-server-components.

    The monitoring is based on a script that pings the server components and writes a document of states in elasticsearch. The data can be visualized with a kibana-dashboard.

    The monitoring consists of following components:

    Windows:

    • insight-status.bat
    • insight-status-servcice.bat

    Linux:

    • insight-status.sh

    • loop.sh

    • Monitoring-Dashboard.json

    Installation:

    • Please create in kibana timebased indexpatttern "insigh-monitoring"
    • Import Monitoring-Dashboard.json
    • Open Dashboard "Monitor"
