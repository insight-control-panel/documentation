Preparation for Windows
❑Extract Insight-middleware-<version>.zip into D:\GiS\Insight
❑The following directory structure is expected:

D:\GiS\Insight

       |

       +----- ElasticSearch

       |

       +----- JRE

       |

       +----- Kibana

       |

       +----- Logstash  

       |

       +----- monitoring

       |

       +----- scripts

 |

       +----- Wildfly

 

Using the Wildfly/TomEE integrated H2 database is not recommended for production usage!

The components ElasticSearch, Kibana and Wildfly use the dedicated JRE to run.

You should not delete the JRE directory even if there is a JRE installed in your system!
